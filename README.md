# 🏥 Hospital Performance Analytics Dashboard (Databricks + SQL)

## 📌 Project Overview

This project presents an end-to-end Healthcare Analytics solution built using Databricks SQL.
The objective was to analyze hospital operations including revenue trends, department performance, doctor productivity, treatment distribution, and patient demographics.

The final output is an interactive dashboard designed for hospital management decision-making.

---

## 🛠️ Tech Stack

* Databricks (SQL Warehouse)
* SQL
* Dashboard Visualization
* GitHub (Version Control)

---

## 📂 Dataset Used

The project integrates multiple healthcare datasets:

* Patients
* Doctors
* Appointments
* Treatments
* Billing

These datasets were joined to create a unified analytical view for reporting and dashboard creation.

---

## 🔄 Data Processing Steps

1. Uploaded CSV datasets into Databricks
2. Cleaned and validated data (null handling, duplicates)
3. Created a unified permanent SQL view (`hospital_data`)
4. Performed aggregations and KPI calculations
5. Built interactive dashboard with filters

---
